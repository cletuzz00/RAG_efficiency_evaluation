Evaluation data for the retrieval benchmark.

Default dataset: **NFCorpus** (BEIR).

Expected files (generated by `scripts/export_beir_to_csv.py`):

- `nfcorpus_test_documents.csv` – columns: `id`, `title`, `text`.
- `nfcorpus_test_queries.csv` – columns: `query_id`, `query`, `expected_answer`, `relevant_doc_ids` (comma-separated document IDs for accuracy).

To generate these from the BEIR NFCorpus dataset:

1. Download NFCorpus into `data/beir/nfcorpus/` (e.g. from [BEIR](https://github.com/beir-cellar/beir) or the public UKP archive).
2. From the project root:

   ```bash
   python scripts/export_beir_to_csv.py --dataset nfcorpus --split test --beir-root data/beir --out-dir data
   ```

Other BEIR datasets (e.g. fiqa, scifact) work the same way with `--dataset <name>`. The active dataset is controlled by `data.dataset` in `configs/experiment_config.yaml`; paths and caches are then derived per dataset.
